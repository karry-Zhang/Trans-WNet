# Trans-WNet: Transferable Wavelet-enhanced neural networks based on domain decomposition for solving the Schrodinger equation

In this paper, we propose a novel approach based on domain decomposition and transferable deep wavelet framework for solving the Schrodinger equation. Specifically, we decompose the computational domain into multiple subdomains, each addressed by an individual neural network. We initially train the neural network across the entire domain, and then use the trained network as the pre-trained model for each subdomain network. Furthermore, we employ wavelet layers to process the input data, extracting features at various scales. Lastly, we conducted experiments on one-dimensional and two-dimensional Schrodinger equations to validate the effectiveness of the proposed approach.
